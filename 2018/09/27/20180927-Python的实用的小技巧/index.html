10. 复制list

如果是刚学习Python的C用户，你可能会想写b = a，但这是错的。在Python中，变量指向的是某个对象的标签。也就是说，按照这种的写法，b和a指向的是内存中的同一个列表，对b操作，就相当于对a操作。所以正确的写法有以下几种：

方法一：b=a[:]。

方法二：b=list(a)。

方法三：使用Python 3的copy()函数，直接复制list，类似a[:]。

方法四：使用copy.deepcopy()。
